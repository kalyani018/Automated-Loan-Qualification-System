# Automated Loan Qualification System

A Java-based console application that evaluates customer eligibility for Personal and Home Loans using predefined banking rules. The system analyzes factors such as age, monthly income, and CIBIL score to generate loan approval decisions.

## Features

* Personal Loan Eligibility Check
* Home Loan Eligibility Check
* Input Validation
* Rule-Based Approval System
* User-Friendly Console Output
* Modular OOP-Based Design

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Eclipse IDE

## OOP Concepts Implemented

* Inheritance
* Method Overriding
* Runtime Polymorphism
* Encapsulation

## Eligibility Factors

* Age
* Monthly Salary
* CIBIL Score

## Project Structure

Loan → PersonalLoan / HomeLoan
